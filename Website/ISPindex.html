Independent Study Project: Creating a Mouse

Vision Statement

Our vision is to learn about circuitry and create a mouse using our knowledge. We will use the tools Solid Edge, Blender, and EasyEDA to create a model and circuitry for the mouse.


Necessary Processes

We will need to first research what type of mouse to make (optical or laser, wired or wireless) and then start creating the circuitry for the mouse with the tools EasyEDA. Additionally, we should create a model of the mouse using Solid Edge and Blender to plan what type of design we are going to use.


Development: Mouse PCB

We will need to create a PCB for the mouse using EasyEDA as part of the development process. Since we have little to none experience with EasyEDA, we will have to first learn how to use it before starting work on the mouse PCB. We can use resources such as the EasyEDA tutorial to learn.

Example mouse PCB

Development: Mouse Model

We will need to first create a model of our mouse to plan what design we are going to be using. We will use Blender and Solid Edge to create our model and then print it out to see how it is. Additionally, we will need to make sure that the model is appropriately sized to fit the PCB that we are designing separately.
