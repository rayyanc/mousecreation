Independent Study Project: Creating a Mouse

Vision Statement

Our vision is to learn about circutry and create a mouse using our knowledge. We will use the tools Solid Edge, Blender, and EasyEDA to create a model and circutry for the mouse.


Necessary Processes

We will need to first research what type of mouse to make (optical or laser, wired or wireless) and then start creating the circutry for the mouse with the tools EasyEDA. Additionally, we should create a model of the mouse using Solid Edge and Blender to plan what type of design we are going to use.
